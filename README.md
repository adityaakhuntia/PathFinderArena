# AI PROJECT 
Pathfinding Arena

Pathfinding Arena is an interactive AI algorithm battle game built with:

- Frontend: HTML, CSS, JavaScript
- Backend: Python

## Current Prototype

- Custom grid builder with start, goal, wall, and erase tools
- BFS, DFS, Greedy Best-First Search, and A* Search
- Race mode with winner highlighting
- Metrics dashboard for nodes explored, path length, and execution time
- Algorithm reasoning feed
- Polished single-page interface served from Python

## Run Locally

```bash
python app.py
```

Then open `http://127.0.0.1:8000`.
